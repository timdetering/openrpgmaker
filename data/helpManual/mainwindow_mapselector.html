[User Interfaces]->[Main Window]->[Map Selector]

This is the map selection area. All of the maps are listed in a tree structure underneath the project.
Left clicking on a map will display it in the map editor section.
Left clicking on an area will display the parent map in the map editor section and allow to edit the selected area.
Right clicking on a map will open the following context menu:
New Empty Map:
Creates a new blank map for editing and opens up the map properties dialog for the newly created map.
New Random Map:
Opens the random map generator dialog for creating a randomly generated map.
New Area:
Creates a new area in the selected map.
Delete Map:
Deletes the selected map (will ask for confirmation).
Map Properties:
Opens the map properties dialog for the selected map.
Edit On-Load Event:
Opens the on-load event editor dialog for editing the maps on-load events (not yet implemented).
