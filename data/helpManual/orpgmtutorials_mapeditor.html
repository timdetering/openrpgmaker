[Open RPG Maker Tutorials]->[Using the Map Editor]

Using the map editor is fairly simple and straight forward.
First click on a map in the map selector box (or an area if you want to edit a map area), and it will be displayed in the map editor.

Next use one of the following four modes (Area Mode if you selected a map area) to edit the map:
Draw Mode
Selection Mode
Tiling Mode
Event Mode
