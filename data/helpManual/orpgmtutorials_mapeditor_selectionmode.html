[Open RPG Maker Tutorials]->[Using the Map Editor]->[Selection Mode]

     The selection mode allows you to copy sections of a map, and paste them elsewhere (even on other maps). Right clicking after drawing a selection will open the following context menu on the right.

NOTE: Some context menu commands may be grayed out depending on the situation.
Copy
Copies the tiles in the current selection (only the current layer is copied), then clears the selection.
Cut
Same as copy, but deletes the selection afterward rather than simply clearing it.
Paste
Pastes the copied selection data into the current map on the current layer. The pasted selection will not be anchored, and will be postitioned in the center of the view.
Delete
Fills the current selection with the tile selection data, then clears the selection.
Duplicate
Same as doing copy then paste.
Clear Selection
Deselects anything that has already been selected.
Anchor Selection
Anchors the floating selection to the current layer.
Detach Selection
Detaches the current selection, which is the same as cutting and pasting it.
     Detached (floating) selections can be moved around the map by pressing the left mouse button anywhere inside the selection, then moving the mouse around. Releasing the left mouse button will place the selection at that location (though it will still be floating, and can therefore be moved again before anchoring it). The difference between a detached (floating) selection and an anchored one, simply look at the outline of the selection, if it is moving clock-wise around the selection then it is detached, otherwise if it is not moving, then it is anchored.
