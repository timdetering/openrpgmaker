[User Interfaces]->[Random Map Generator]->[World Map Generator]


Name
The name for the generated map.
Map Size
The size of the generated map.
Num Continents
The number of different continent terrain to use (between 1 and 5).
NOTE: This does not indicate the number of continents to be placed.
Water Tile
The tile used for placing water surrounding the continents.
Blank Tile
The tile used to fill in the hero layer and upper layer (should be filled with the transparent color magenta).
Continent -> Number
The continent currently being edited (between 1 and "Num Continents").
Continent -> Num Terrain
The number of different terrain tiles to be placed on the continent (between 0 and 6).
Continent -> Tile
The tile used for the continent.
Continent -> Amount
Roughly how many continents of this type to place.
Continent -> Size
The rough size of each continent of this type.
NOTE: Placing a lot of medium sized continents could result in one giant continent depending on the size of the map.
Terrain -> Number
The terrain currently being edited (N/A if the current continent has 0 terrain).
Terrain -> Tile
The tile used for this terrain (will be placed on the hero layer).
Terrain -> Amount
Roughly how many patches of this terrain to place on the continent.
Terrain -> Size
The rough size of each patch of terrain.
