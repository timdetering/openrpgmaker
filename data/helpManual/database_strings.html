[User Interfaces]->[Database]->[Strings Tab]

The "Strings Tab" is split into three pages, and each string is fairly self explanatory. So their will only be listed below what variables the strings allow, and what $(X) does if allowed.

Each string may allow one or more of the following variables:
$(CURRENCY): This will be replaced with the "Currency" string on Page 2 under "Other Strings".
$(CHARACTER): This will be replaced with the name of the character being referred to.
$(X): This gets handled differently depending on which string it appears in.
